# Spring 핵심 원리

## 📚 강의 개요

스프링 프레임워크의 핵심 원리와 객체 지향 설계를 학습합니다.

## 📖 학습 내용

### 1. 객체 지향 설계와 스프링
- SOLID 원칙
- 좋은 객체 지향 설계의 5가지 원칙
- 스프링의 탄생 배경

### 2. 스프링 핵심 원리 이해
- 의존성 주입(Dependency Injection)
- 제어의 역전(Inversion of Control)
- DI 컨테이너

### 3. 스프링 컨테이너와 빈
- 스프링 컨테이너
- 스프링 빈
- 빈 스코프
- 빈 생명주기

### 4. 컴포넌트 스캔과 의존관계 자동 주입
- 컴포넌트 스캔
- @Autowired
- 다양한 의존관계 주입 방법

### 5. 빈 스코프
- 싱글톤 스코프
- 프로토타입 스코프
- 웹 스코프
