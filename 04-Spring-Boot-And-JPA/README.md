# Spring Boot와 JPA

## 📚 강의 개요

JPA(Java Persistence API)의 기본 개념과 Spring Boot를 활용한 실무 애플리케이션 개발을 학습합니다.

## 📖 학습 내용

### 1. JPA 소개
- SQL 중심 개발의 문제점
- JPA와 ORM
- 영속성 컨텍스트

### 2. 엔티티 매핑
- 객체와 테이블 매핑
- 필드와 컬럼 매핑
- 기본 키 매핑

### 3. 연관관계 매핑
- 단방향 연관관계
- 양방향 연관관계
- 연관관계의 주인
- 다대일, 일대다, 일대일, 다대다

### 4. 상속관계 매핑
- 조인 전략
- 단일 테이블 전략
- 구현 클래스마다 테이블 전략

### 5. 프록시와 연관관계 관리
- 프록시
- 즉시 로딩과 지연 로딩
- 영속성 전이(CASCADE)
- 고아 객체

### 6. 값 타입
- 기본값 타입
- 임베디드 타입
- 값 타입 컬렉션

### 7. 실전 예제
- 도메인 모델 설계
- 엔티티 클래스 개발
- 애플리케이션 구현
