# Spring MVC

## 📚 강의 개요

스프링 MVC 패턴과 웹 애플리케이션 개발을 학습합니다.

## 📖 학습 내용

### 1. 웹 애플리케이션 이해
- HTTP 요청과 응답
- 서블릿
- JSP와 MVC 패턴

### 2. Spring MVC 구조
- DispatcherServlet
- Handler Mapping
- Handler Adapter
- ViewResolver

### 3. 요청 매핑과 HTTP 메서드
- @RequestMapping
- @GetMapping, @PostMapping
- 경로 변수와 쿼리 파라미터

### 4. HTTP 요청 데이터 처리
- @RequestParam
- @ModelAttribute
- @RequestBody

### 5. HTTP 응답 처리
- 정적 리소스
- 뷰 템플릿
- HTTP 메시지 바디

### 6. 타임리프
- 타임리프 기본 문법
- 반복과 조건부 렌더링
- 폼 처리

### 7. 검증과 예외 처리
- Validation
- BindingResult
- 예외 처리

## 📝 학습 자료

- 강의 노트
- 웹 애플리케이션 프로젝트
- 실습 코드
