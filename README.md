# 김영한님의 강의 학습 자료 📚

인프런(Inflearn)에서 강의를 진행하는 김영한님의 강의를 학습하며 정리한 자료입니다.

## 📖 강의 목록

### 1. Java 기초
**디렉토리**: `01-Java-Basic/`
- Java 프로그래밍 기초
- 객체 지향 프로그래밍
- Java 핵심 문법

### 2. Spring 핵심 원리
**디렉토리**: `02-Spring-Core/`
- 스프링 프레임워크 핵심 원리
- 객체 지향 설계와 SOLID 원칙
- 의존성 주입(DI)과 제어의 역전(IoC)
- 스프링 컨테이너와 빈

### 3. Spring MVC
**디렉토리**: `03-Spring-MVC/`
- 스프링 MVC 패턴
- HTTP 요청과 응답
- 뷰 템플릿과 타임리프
- 검증과 예외 처리

### 4. Spring Boot와 JPA
**디렉토리**: `04-Spring-Boot-And-JPA/`
- JPA 기본 개념
- 엔티티 매핑
- 연관관계 매핑
- 프록시와 연관관계 관리
- 값 타입

### 5. Spring Data JPA
**디렉토리**: `05-Spring-Data-JPA/`
- Spring Data JPA 활용
- 쿼리 메서드
- 페이징과 정렬
- 확장 기능

### 6. QueryDSL
**디렉토리**: `06-QueryDSL/`
- QueryDSL 기본 문법
- 동적 쿼리
- 성능 최적화
- 실무 활용
